# ``BbAppContentSourceWordPressBase``

Base WordPress content source functionality.

## Overview

The BbAppContentSourceWordPressBase library provides the foundational implementation for WordPress-based content sources. It includes core WordPress API communication logic, data transformation utilities, field definitions for WordPress API responses, and common functionality shared between different WordPress content source variants. This base implementation can be extended by specific WordPress provider implementations.
